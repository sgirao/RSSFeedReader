# Quickstart: MVP RSS Reader

## Prerequisites
- .NET SDK 8 or later
- A modern browser

## Run the MVP
1. Start the backend API.
2. Start the Blazor frontend.
3. Open the frontend URL in the browser.
4. Enter a feed URL and submit it.
5. Confirm that the subscription appears in the list.

## Expected Result
A user can add a subscription and see it appear in the UI without requiring persistence or feed-fetching features.
